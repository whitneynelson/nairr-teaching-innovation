# Module 07: AI for Grading & Attendance

**Optional, self-paced**; not scheduled as a live session this cohort. This module isn't part of the live agenda, but the content is here if you want to work through it on your own, before or after the accelerator.

## Ground rules; read this first

Never paste identifiable student data into an AI-powered coding assistant without de-identifying it first: name, ID, email removed or replaced with a placeholder. This is a hard line, not a preference.

## Step 1: Feedback prompt templates

Draft reusable feedback prompt templates for common assignment issues into `course-toolkit/grading-and-attendance/feedback-prompts.md`. Test at least one against de-identified sample data.

## Step 2: Attendance pattern template

Draft `attendance-pattern-template.md`; a way to spot engagement trends without exposing individual student records in the process.

## Checkpoint

<ul class="checklist">
<li>Ground rules documented in <code>feedback-prompts.md</code></li>
<li>At least one feedback prompt template drafted and tested on de-identified sample data</li>
<li><code>attendance-pattern-template.md</code> drafted</li>
</ul>
