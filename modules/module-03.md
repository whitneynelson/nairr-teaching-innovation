# Module 03: Draft Your Syllabus

*Feeds: Day 2, 12:00–12:30 PM*

<div class="card">

**In plain terms:** this session produces a *draft skeleton*, not a finished syllabus — some sections get intentionally left as placeholders because the content they need (like HPC access language) hasn't been covered yet.

</div>

## Step 1: Start from what you have

- **Have an existing syllabus:** open the one you uploaded pre-event. You're revising, not starting blank.
- **Starting from scratch:** use the model syllabus provided. Adapt it with Claude — e.g. *"Adapt this model syllabus for a [course topic] course with these learning outcomes: [paste from Module 02]."*

## Step 2: Pull in what you already built

- Learning outcomes → summarize from `course-toolkit/learning-outcomes-map.md` (public-safe version only — no rubric detail)
- Schedule → link to `course-site/schedule.md`

## Step 3: Leave placeholders where content isn't ready yet

Mark these clearly so they don't get forgotten:
```
[NAIRR/TACC ACCESS LANGUAGE — completed Day 4]
[GRADING POLICY SUMMARY — completed Day 4]
```

## Step 4: Check against your institution's policy checklist

Compare against the checklist you uploaded pre-event. Common gaps: accommodations statement, academic integrity/AI-use policy, attendance policy.

## Checkpoint

<ul class="checklist">
<li><code>course-site/syllabus.md</code> has a draft course description, outcomes summary, and schedule link</li>
<li>Placeholders clearly marked for sections not yet built</li>
<li>Checked against your institution's policy checklist — gaps noted</li>
</ul>
