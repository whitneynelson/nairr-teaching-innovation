# Module 10: Reuse This Template

*Feeds: Day 5, 10:15–11:00 AM (shared block with Module 09)*

<div class="card">

**In plain terms:** the actual "duplicate the repo" step takes about 5 minutes — most of the value here is in thinking through what carries over.

</div>

## Step 1: Confirm both repos are set as templates

**Settings** → check **"Template repository"** on both `course-site` and `course-toolkit`.

## Step 2: Walk through "Use this template" once

The green button produces a clean copy, no shared commit history, ready to rename.

## Step 3: What carries over vs. what doesn't

- **Carries over cleanly:** repo structure, rubric templates, the AI-use rubric, this Builder Guide walkthrough itself.
- **Needs rebuilding each time:** learning outcomes, curriculum map, actual module content, exam questions.

## Step 4: Note it in your 90-day plan

Flag which toolkit artifacts you'd reuse as-is vs. rebuild.

## Checkpoint

<ul class="checklist">
<li>Both repos confirmed as GitHub template repos</li>
<li>Clear on what carries over vs. rebuilds for next use</li>
</ul>

This is the last module — congratulations, both repos are built and your course site is live.
