# nairr-teaching-innovation

**This repo is PUBLIC.** It's the front door for the NAIRR Teaching Innovation Accelerator — published via GitHub Pages.

## What's in here

| File | Purpose |
|---|---|
| `_config.yml` | GitHub Pages / Jekyll site config |
| `index.md` | Program overview |
| `agenda.md` | Full hour-by-hour schedule, pre-event through departure |
| `modules.md` | What participants build, module by module |

This is a marketing/informational site for the accelerator program itself — it's separate from `course-site`, which is the template each participant uses to build their own course during the week.
