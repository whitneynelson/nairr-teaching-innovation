# NAIRR Teaching Innovation Accelerator

A week-long accelerator (July 27–31) for instructors who want to build courses with real HPC-based assignments — without needing to already know how to code.

## Who this is for

Open to instructors across every discipline and skill level:

- **If you code**, Claude Code speeds up writing and debugging the scripts your students will run on HPC.
- **If you don't code**, Claude Code writes those scripts *for* you, in plain English, so you can still build and understand HPC-based assignments without becoming a programmer in the process.

Everyone leaves the week with two things: a public course site and a private instructor toolkit, both built and ready to publish.

## What you'll build

Across five days, you'll work through a guided, step-by-step course-build process — see [Modules](modules.md) for the full breakdown, and [Agenda](agenda.md) for the hour-by-hour schedule.

By the end of the week you'll have:
- A published, public course site — syllabus, schedule, and one fully built module
- A private grading toolkit — rubrics, an exam scaffold, and a grading scale
- A 90-day plan for using what you built beyond this week

## Before you arrive

- Create your TACC account and set up MFA
- Install Git and an IDE (VS Code or Antigravity)
- Create a free GitHub account
- Bring your current syllabus (or use the model syllabus provided) plus your institution's syllabus policy checklist

Full pre-event checklist and timing: see the [Agenda](agenda.md).
