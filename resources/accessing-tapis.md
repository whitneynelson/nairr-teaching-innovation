# Accessing the Morehouse Supercomputing Facility via TAPIS

<div class="card">

Tapis is the layer that lets you (and your students) run jobs on the supercomputer without managing the underlying infrastructure directly; it's the interface between "I want to run this script" and the actual hardware. This page is the full path, request to first run.

</div>

## 1. Request access

Request your account at [bpccenter.org/current-programs/cbpc-hpc-resources](https://www.bpccenter.org/current-programs/cbpc-hpc-resources), using your .edu email address.

## 2. Wait for approval

**Approval can take a few business days**; the same caveat as everywhere else this program mentions account setup, which is why this step is due well before the accelerator starts, not the week of. If you haven't heard back within a week, follow up with your program contact rather than assuming it's stuck.

## 3. Log in and connect via Tapis

Once approved, you'll log in through your browser and, for the hands-on sessions this week, through Tapis itself; this is covered live in Module 00–01 and the HPC Acclimation sessions (Day 2–3), so you don't need to work this out solo beforehand. If your specific program setup requires an SSH client on Windows, PuTTY or the Git Bash terminal that came with Git for Windows both work.

## 4. Walk through a runnable example

Once you're logged in, the fastest way to get oriented is to work through an actual example rather than reading about the concepts in the abstract:

- [Accessing Morehouse TAPIS (walkthrough doc)](https://docs.google.com/document/d/1-44pAXIM6qfDUw5ha-RQYFyuLKGypIQONGJR-6LWaWA/edit?tab=t.0#heading=h.mci92mcba1x1); a step-by-step walkthrough you can follow along with directly.
- [TAPIS workflows reference](https://ashleyscruse.github.io/nairr-research/03-Compute/05-tapis-workflows/); the underlying reference material the walkthrough draws from, useful to come back to once you're building your own assignment.

## Further hands-on training (optional)

If you want more practice beyond what this week covers, [Life Sciences ML at TACC](https://life-sciences-ml-at-tacc.readthedocs.io/en/latest/section2/overview.html) is a solid self-paced option for going deeper on HPC workflows; not required for the accelerator, just useful if you want extra reps.
